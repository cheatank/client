# CheaTank Client

## Library

- [CheaTank common](https://github.com/cheatank/common)
- [KorGE](https://korge.org)
- [Ktor](https://ktor.io)

## Script

### Run app

```shell
# Browser
./gradlew runJs

# JVM
./gradlew runJvm
```

### Format code

```shell
./gardlew formatKotlin
```

### Check dependencies version

```shell
./gradlew dependencyUpdates
```
